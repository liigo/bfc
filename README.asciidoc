= bfc

A Brainfuck compiler written in Rust.

Build: `cargo build`

Compile and execute Brainfuck source code:

```
./target/bfc "++++++ [ > ++++++++++ < - ] > +++++ ."
```
will print A.

```
./target/bfc "++++++++++[>+++++++>++++++++++>+++>+<<<<-]
>++.>+.+++++++..+++.>++.<<+++++++++++++++.
>.+++.------.--------.>+.>."
```
will print Hello World!

```
./target/bfc ", [ > + < - ] > ."
```
will echo any char you input.
